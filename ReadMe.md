# notes-cli-rust

## Запуск
`cargo run`

## Команды
- add
- show
- exit

## Пример работы программы
```
Введите команду (add, show, exit): add
Введите название заметки: Создать программу для заметок
Введите текст заметки: должны быть три команды: add, show, exit
Заметка "Создать программу для заметок" добавлена
Введите команду (add, show, exit): add
Введите название заметки: Запушить на github
Введите текст заметки: Не забудь про ReadMe
Заметка "Запушить на github" добавлена
Введите команду (add, show, exit): show
Список заметок:
---Заголовок: Создать программу для заметок---
Текст: должны быть три команды: add, show, exit

---Заголовок: Запушить на github---
Текст: Не забудь про ReadMe

Введите команду (add, show, exit): exit
Программа завершена.
```
